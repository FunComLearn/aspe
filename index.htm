Internet Overview

 Internet

       To access data over the web, the Internet is described as an information major roadway.

     In several ways, however, it can be described as follows:

·  The Internet is a global web of interconnected networks of computers worldwide.

·  The basic Internet Protocol (TCP/IP) is used on the Internet.

·  A specific IP address identifies every device on the Internet.

·  The IP address is a specific series of numbers that determines a device location, such as 110.22.33.114.

·  A special Domain Name Server (DNS) device is used to describe the IP address so that the user will be able to find the computer by its name.

·  For example, a DNS server resolves the name http://www.tutorialspoint.com to a specific IP address, so that the device on which this website is stored can be uniquely identified.

·  The Internet is usable to all users worldwide.

 

 

 

   

 Evolution

 

 

 

 

 

 

 

 

 

 

 

 The concept of Internet was originated in 1969 and has undergone several technological & Infrastructural changes as

   discussed below:

·  The history of the Internet was created from the Advanced Research Project Agency Network idea (ARPANET).

·  ARPANET was formulated by the Department of Defense of the United States.

·  There were just four nodes originally, officially called Hosts.

·  By the time, the Internet offered a means for publishing and viewing information over the internet with the invention of new technologies such as TCP/IP protocols, DNS, WWW, browsers, scripting languages, etc.

·  In1972, with 23 nodes based in different countries, the ARPANET spread across the globe and therefore became widely acknowledged as the Internet.

Intranet Overview

 Intranet

  The intranet is acknowledged as a private computer network within an organization with its own firewall and server.

   In addition, we can describe intranets as:

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

§ Intranet is a system wherein multiple PCs are interconnected to each other via a network. Intranet PCs are not readily accessible outside its intranet world - wide.

 

§ A unique IP address identifies every computer on the Internet.

 

§ Each intranet computer is so often recognized by an IP address that is common to the computers on that intranet.

 

§ Generally, each corporate entity or institution seems to have its own intranet network and intranet computers can be accessed by members/employees of that company.

Extranet

 Extranet pertains to an organization's network, exploiting the internet in a monitored means of communicating to strangers. It involves linking companies with their customers and suppliers and thus makes it possible to engage in a coordinated environment.

 

Implementation

 Extranet is established as a Virtual Private Network (VPN) since it connects to business enterprises using the internet and information security is so often threatened. In public infrastructure, VPN provides a stable network (Internet). By introducing an external security layer to the TCP/IP protocol, VPN uses the Internet Protocol Security Architecture (IPSEC) protocol to provide secured transactions.

 

Benefits

The DNS scheme was launched to come up with a solution.

 

The Domain Name System aims to solve the location of the host name. A hierarchical network framework and a transmitted IP address database and corresponding names are often used.

 

IP Address

 A specific logical address registered to a computer over the network is the IP address. The following properties manifest an IP address:

§ Two components comprise of an IP address: the host component and the network component.

§ The IP address is 4 bytes long (32 bits).

§ Each one of the 4 bytes is characterized by a number, segregated by dots, from 0 to 255. For instance, 137.170.4.1244

§ The unique address appointed to each host on the Internet is indeed an IP address.

 

 The IP address is a 32-bit number, while domain names, on the other hand, are easy to recall.

Cable TV Internet Connection

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

The Internet connection for cable TV is supported by cable TV lines.

It incorporates a coaxial cable that can transmit data mostly at higher speeds than ordinary telephone lines. To activate this platform, a cable modem should be used, which is made available by the cable operator. There are two connections to the cable modem: one for Internet access and another one for cable TV signals. Although cable TV internet connections combine with a number of consumers a fixed amount of bandwidth, the rate of data transmission actually depends on the number of consumers accessing the internet at the same time.
